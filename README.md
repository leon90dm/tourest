### Run Locally

To run **TourSnap** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/TourSnap.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/TourSnap.git
```
